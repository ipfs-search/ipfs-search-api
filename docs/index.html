get /search
Search for files and directories. (searchGet)
By default, the score for items with a more recent <code>last-seen</code> date are positively boosted while 'unnamed' items (without a <code>reference</code> or a <code>title</code>) are negatively boosted.

Query parameters

q (required)
Query Parameter — Search string query, based on Elasticsearch's <a href="https://www.elastic.co/guide/en/elasticsearch/reference/current/query-dsl-query-string-query.html#query-dsl-query-string-query">Query string query</a> syntax.
type (optional)
Query Parameter — Resource type. Omit to return all types.
page (optional)
Query Parameter — Page number. default: 0

Return type
